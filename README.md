# TrendPredict 
TrendPredict is a real-time analytics dashboard built with Tableau that analyzes Twitter/X trends, performs sentiment analysis, and predicts emerging topics using data-driven insights.

## Features
- Real-time Twitter/X trend monitoring
- Sentiment analysis (positive / neutral / negative)
- Predictive trend forecasting
- Interactive Tableau dashboards
- Actionable insights for decision-makers

## Tech Stack
- Tableau (Tableau Public)
- Twitter/X API (Free Tier)
- Python (Tweepy, Pandas, TextBlob)

## Use Case
Helps marketers, analysts, and strategists identify trending topics early and take timely action.
